
# SPARK
Repository has necessary for this registry informations from spark exploration

## Commands for run
Commands to execute with spark,  Spark comes with several sample programs. Scala, Java, Python and R examples

|Language		 |Command run					  |Command run project or file		   |  
|----------------|--------------------------------|------------------------------------|  
|**PYTHON** 	 |`$./bin/pyspark`                |`./bin/spark-submit [file.py]`      |  
|**R** 			 |`$./bin/sparkR`                 |`./bin/spark-submit [file.r]`       |  
|**JAVA** 		 |`$./bin/run-example`            |`./bin/run-example <class> [params]`|  
|**SCALA** 		 |`$./bin/spark-shell`            |`./bin/run-example <class> [params]`|
